Project Overview

In this project, we implement a real-time cloth simulator. We do this by first constructing our cloth mesh through point masses and springs. We then apply the appropriate forces to our mesh and calculate the position displacements that consequently occur, while also considering constraints that we put on the mesh so that it does not continuously move too far out of place, when our formulas do not converge. Next, we account for the cloth's collision with not only other objects, such as a plane or a sphere, but also with itself.
